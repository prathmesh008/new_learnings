## 2026-05-08 — Redis data structures

Been reading about Redis today and damn, Hashes finally clicked. I always just crammed JSON blobs into regular string keys like `SET user:123 '{...}'` but that's so inefficient for updates. If you just need to change one field, a Hash lets you `HSET user:123 email "new@email.com"` without fetching and re-saving the whole damn thing. Felt kinda stupid for not realizing it sooner, ngl.
