## 2026-05-16 — React useCallback vs useMemo

Okay, finally sorta clicked with `useCallback` vs `useMemo` today. Been re-reading that React docs page for the third time this week. The big thing that threw me off was thinking they both just "memoized stuff." Nah, `useCallback` is literally just for *functions* man, like `const doStuff = useCallback(() => { console.log('hi') }, [])`. And `useMemo` is for values, like a computed object or a heavy calculation. Makes so much more sense now why they're separate hooks.
