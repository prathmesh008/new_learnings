## 2026-04-12 — Redis data structures

Okay so today it really clicked how useful Redis Sorted Sets are. Like for leaderboards, duh, but I was thinking about how it simplifies other stuff too. `ZADD` lets you assign a score, so you can just `ZRANGE` by score for top players, or even for time-based stuff if the score is a timestamp. Before I was overthinking with separate lists or trying to sort in the app, this is way cleaner. `ZADD game:scores 1234 "player_alice"`. So much quicker.
