## 2026-05-05 — CSS specificity

Okay so specificity in CSS, wow. Been reading about it for ages but today it just clicked how much an ID selector actually overrides everything else. Like, even if you put a class rule *after* it in the stylesheet, if there's an `#id` on the element, that thing wins. So all my `.card` styles were getting totally stomped by `#main-promo` or something. Explains so much frustration from last week, ha. Gotta remember that scoring system for next time.
